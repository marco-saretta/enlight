# System Requirements

## Hardware Requirements
- Minimum: 4 GB RAM, 1 CPU core
- Recommended: 8 GB RAM, 2 CPU cores

## Software Requirements
- Operating System: Linux, macOS, Windows
- Python >= 3.8
